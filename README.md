# 2023 데이터 청년 캠퍼스
빅데이터 분석 기반 지능 소프트웨어 과정 (한국데이터산업진흥원)

---

## 📌 프로젝트 개요
- **주제**: 2050 탄소중립을 위한 경기도 산림재구성  
- **기간**: 2023.07.24 ~ 2023.08.31 (6주)  
- **참여 인원**: 팀 프로젝트 (총 3명)  
- **역할**:  
  - 데이터 수집 및 전처리  
  - 최적 군집 개수 산출을 위한 **클러스터링**  
  - **PCA 분석** 수행  
  - **Prophet 기반 행정동 인구수 예측**  
  - 인구 증가에 따른 **CO(일산화탄소), NO₂(이산화질소) 농도 분석**
